## 測試/開發
因為此專案需利用外部資源，故無法直接在瀏覽器裡打開[`index.html`](./src/index.html)進行測試或開發，若是要的話須使用像是[`http-server`](https://www.npmjs.com/package/http-server/)或類似的軟體。以[`http-server`](https://www.npmjs.com/package/http-server/)要跑：

```
npx http-server src
```

即可。
